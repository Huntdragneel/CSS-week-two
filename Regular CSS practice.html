List of Style Modifications

Lists of actvie styles applied in this section.

  • color changed the text color
  • background-color changed the background color
  • border-color changed the border color but border-style has to be defined in order to have an impact
  • border-radius makes the edges of the border curved
  • text-shadow gives the desired text a shadow of your color of chosing (applied to the text above the list)
  • font-size changes the size of the text (applied to the text above the list)
  • both the main tag and section tag for this page have a CSS class that was created in the opening tag called bordered. It had the border as a solid, curved, with a width of 20px, and black
  • id style changes override tag changes
  • all p and h1 tags have text-align: center
